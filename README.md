# Our Story

A private couple's memory site — timeline, daily journal, photos & videos, and more.

- **web/** — Vite + React + TypeScript + Tailwind (mobile-first, light/dark). Deploy on Cloudflare Pages.
- **server/** — Node + Express + Prisma (PostgreSQL/Neon), media on Cloudflare R2. Deploy on Render.

Private by design: no public sign-up, two seeded accounts only.
